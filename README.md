## Descripción
*El trabajo practico integrador final consistió en la creación de un CRUD para las relaciones entre dos tablas: Producto y CodigoBarras, con una relación 1-1.
![[Diagrama.png]]
El patrón de diseño utilizado fue DAO, separando la lógica individual de cada tabla para interactuar directamente con la base de datos.
La base de datos utilizada fue también, en parte, la base de datos que se utilizo para la cátedra Base de Datos. En ella se utilizo MySQL como motor y registro el proceso de creación, medidas de seguridad, formas normales a través de Worbench.
Para la conexión entre JAVA y la base de datos se incluyo a la libreria mysql-connector-j-8.0.33
*![[conexionBD.png]]

*El código fuente de la creación de la base de datos se incluirá en el repositorio al final de este documento, junto con otros archivos de interés *
